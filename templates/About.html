About This AQI Project

Understanding Air Quality Prediction Using Machine Learning.


The AQI Prediction Using Machine Learning project predicts the Air Quality Index based on pollutant data, including PM2.5, PM10, NO₂, SO₂, CO, and O₃. A trained machine learning model analyzes these inputs and estimates the AQI. Built with Python and Flask, the system provides accurate, fast, and user-friendly air quality prediction.

What is This Project?

The Air Quality Prediction System is a Machine Learning-based web application designed to predict the Air Quality Index (AQI) using environmental and air pollution data...

Health Impact

Air pollution has a significant impact on human health and is one of the leading environmental risk factors worldwide...

Machine Learning Model

Algorithm

Random Forest is a Machine Learning algorithm that uses many decision trees to predict AQI. It analyzes pollution data such as PM2.5, PM10, NO₂, SO₂, CO, and O₃ to provide accurate and reliable air quality predictions.

💹Performance

R² score of 0.99687

indicating excellent prediction accuracy. This high score shows that the model predicts the Air Quality Index (AQI) very accurately and provides reliable results for the given pollution data.

Feature Used

PM2.5, PM10, NO₂, SO₂, CO, and O₃
